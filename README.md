# 테트리스 (교육용)

HTML, CSS, JavaScript만 사용하는 브라우저 테트리스 프로젝트입니다.

## 실행 방법

1. 이 폴더를 연다.
2. `index.html` 파일을 더블클릭하거나, 브라우저로 드래그한다.
3. **시작** 버튼을 누른 뒤 키보드로 조작한다.

## 조작법

| 키 | 동작 |
|----|------|
| `ArrowLeft` / `ArrowRight` | 좌우 이동 |
| `ArrowDown` | 한 칸 빠르게 내리기 |
| `ArrowUp` | 시계 방향 90° 회전 |
| `Space` | 즉시 낙하 (hard drop) |
| `Shift` (좌/우) | 보관함에 블록 보관·교환 (조각당 1회) |

## 기능

- **다음 블록**: 게임 보드와 점수 UI 사이에 다음 3개 블록 미리보기
- **보관함**: Shift로 현재 블록을 보관하거나 보관 블록과 교환
- **고스트**: 블록이 떨어질 위치를 반투명으로 표시
- **가속**: 점수가 오를수록 낙하 간격이 짧아짐 (200점마다 25ms 단축, 최소 120ms)
- **회전 유예**: 회전 후 일정 시간(1.2초) 동안 자동 고정되지 않음 (T-스핀 연습용)
- **게임 오버**: 맨 윗줄(row 0) 위로 블록이 배치되는 순간 종료

## 점수 규칙

- 가득 찬 줄 1줄당 **100점**

## 게임 오버

- 블록을 고정할 때 보드 **맨 윗줄 위**(숨겨진 스폰 영역)로 넘어가면 게임 오버
- 맨 윗줄이 채워지기 전에 스폰 불가로 종료되지 않음

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 화면 구조 |
| `style.css` | 스타일 |
| `script.js` | 게임 로직 |
